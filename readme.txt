# badnick plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

badnick plugin (v1.9) for B3

Installation:

1. Place the badnick.py in your ../b3/extplugins and the 
badnick.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="badnick" config="@b3/extplugins/conf/badnick.ini"/>

3. Run the contact SQL script (badnick.sql) on your B3 database
